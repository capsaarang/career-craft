# CareerCraft ✦

> AI-powered resume tailoring, application tracking, community insights, and interview prep — all in one platform.

Built by [Saarang Govinda Rajan](https://capsaarang.github.io/myportfolio/) · Powered by Anthropic Claude API + Supabase

## Features
- **✦ Tailor Resume** — 3-panel input: original resume + LaTeX template + JD → perfectly tailored LaTeX output in seconds
- **⊞ Application Tracker** — track every application with status, interview dates, notes, CSV export
- **❋ Community Feed** — share tips, celebrate wins, ask questions, upvote helpful posts
- **◈ Interview Prep** — AI-generated interview questions organized by category with answer tips
- **⚡ Admin Panel** — full visibility into all users, applications, and community activity (admin only)
- **Resume Library** — save multiple resume versions (TPM, BA, Consulting) and load with one click
- **Motivational quotes** — rotating quotes to keep you motivated through the job search

## Tech Stack

- React + Vite
- Supabase (auth + database)
- Anthropic Claude API
- Netlify (hosting + serverless functions)

## Setup

See SUPABASE_SETUP.md for full database setup instructions.
